Social Icons Widget to displays links to social sharing websites. Supports more than 100 sites and includes 150 icons.

== Description ==

**Social Icons Widget** helps you to add quickly icons with links to your profiles from different social networks. The plugin includes socicon set, which gives you the possibility to use the widget for your social networks.

== Installation ==

1. Activate Plugin
2. Go to Appearance/widgets
